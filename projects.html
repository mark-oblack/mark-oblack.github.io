Web & UX

Memory Game

A memory game that challenges a user to match 8 pairs of cards. The functionality is built with vanilla JS and uses DOM properties to change CSS classes applied to a selected HTML element.

AEO Client Project

A class project where I and a group of students were tasked with creating a contextualized app experience for American Eagle Outfitters customers. My group targeted a painpoint of not being able to try on clothes when shopping online.

Wikipedia Viewer

A web app that takes a user's input and returns search results from the Wikipedia API using an AJAX request. A user can also navigate to a random Wikipedia article with a click of a button.

Flint Clothing Website

A class project where I tasked with designing a user interface for a sportswear clothing line that includes apparel for men, women, and children. The goal of this website is to alleviate common pain points people experience when shopping online.
