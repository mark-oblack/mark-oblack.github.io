Projects

Local Weather App

A local weather app that obtains a user's location via geolocation and pulls weather data from the Dark Sky API. A pure CSS loading icon is used on the landing page and flexbox is used for the page layout.

AEO Client Project

A class project where we were tasked with creating a contextualized app experience for American Eagle Outfitters customers. My group targeted a painpoint of not being able to try on clothes when shopping online.

Random Quote Machine

A JavaScript app that displays a quote from an array based on a randomly generated number. The quote is also converted to a Twitter URL that a user can Tweet out with the click of a button.

Pixel Art Maker

A web app that allows a user to define a grid size and design pixel-based art. Users can also change the grid background and gridline colors. Functionality is accomplished with JavaScript and jQuery.

Memory Game

A memory game that challenges a user to match 8 pairs of cards. The functionality is built with vanilla JS and uses DOM properties to change CSS classes applied to a selected HTML element.

Wikipedia Viewer

A web app that takes a users input and returns search results from the Wikipedia API using an AJAX request. A user can also navigate to a random Wikipedia article with a click of a button.
